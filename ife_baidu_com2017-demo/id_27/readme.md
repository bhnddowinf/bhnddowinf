# 使用CSS实现折叠面板进阶

# 任务目的
- 深入理解html中radio的特性
- 深入理解CSS选择器以及伪元素的使用

#任务描述
使用input的radio单选框特性结合CSS中的伪元素实现bootstrap中折叠面板(点击查看样例)，要求样式一致。
[官網 demo] {http://v3.bootcss.com/javascript/#collapse-example-accordion}

# 任务注意事项
当其中一个面对折叠时，其他面板需要隐藏
只能使用HTML，CSS，不允许使用JavaScript
注意测试不同情况，尤其是极端情况下的效果
有能力的同学在面板折叠或者展开时添加动画效果
